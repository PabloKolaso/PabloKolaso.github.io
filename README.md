# Pablo Kolaso — Portfolio

Personal developer portfolio site, live at **[PabloKolaso.github.io](https://PabloKolaso.github.io)**.

## What it is

A single-file portfolio that showcases my projects and skills. It:

- Fetches all public repos live from the GitHub API
- Highlights pinned projects (AniLibria-Stremio, Anime_DUBOMBI) with full context
- Auto-detects languages/skills from repo data
- Supports dark/light mode with preference persistence
- Works fully responsive on mobile and desktop

## Stack

Pure HTML + CSS + vanilla JS — no frameworks, no build step, no dependencies.

## Running locally

Just open `index.html` in a browser. No server needed.
